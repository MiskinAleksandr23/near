# Adversarial audit of the linear affine-modification theorem

Throughout, `k` is algebraically closed of characteristic zero.  This note
audits both `linear_affine_modification_theorem.md` and
`linear_coefficient_theorem.md`.  The two files contain the same argument.

## 1. Verdict on the exponent-one theorem

The proof of

```
f=u x+G(u,v,w),  k[x,u,v,w]=B[f]  =>  f is a coordinate
```

is correct.  The checks below record the hypotheses actually used and close
the shortcuts in the current presentation.

### 1.1 Stable descent

The identification `B = R/(f)` is legitimate: under the given internal
polynomial presentation `R=B[f]`, evaluation at the polynomial variable `f`
has kernel `(f)`.  In particular, `B` is a finitely generated domain.

Regularity descends from `B[T]` to `B`.  For example, at a prime `p` of `B`,
localize `B[T]` at the prime `pB[T]`; comparison of dimension and embedding
dimension shows that `B_p` is regular.  Hence `B` is normal.  For a normal
noetherian domain the standard map

```
Cl(B) -> Cl(B[T])
```

is an isomorphism.  Thus `Cl(B)=0` because `B[T]=R` is factorial, and `B` is
a UFD.  Finally `B^*=B[T]^*=R^*=k^*`.  There is no missing geometric
factoriality hypothesis here: ordinary factoriality over `k` is all that the
divisor argument uses.

### 1.2 The divisor calculation

Suppose first that `g_0=G(0,v,w)` is nonconstant and squarefree, with distinct
irreducible factors `p_i`.  In

```
B=k[x,u,v,w]/(ux+G)
```

the ideals `P_i=(u,p_i)` are precisely the height-one primes above `(u)`.
Since

```
B/(u)=k[x,v,w]/(g_0)
```

is generically reduced along every component, the valuation of `u` at each
`P_i` is one.  Consequently

```
div_B(u)=sum_i P_i.
```

If `P_i=(h_i)`, then `h_i` becomes a unit in

```
B_u=k[u,u^(-1),v,w],
```

whose unit group is `k^* u^Z`.  Thus `h_i=c u^n` in the common fraction
field.  Taking the valuations at every `P_j` gives

```
delta_ij=n.
```

This excludes more than one component.  The equality in the fraction field,
not merely an equality up to an unspecified unit of `B_u`, is justified by
the displayed description of `B_u^*`.

The proof that `g_0` is squarefree is also sound.  A repeated irreducible
factor would make `g_0` and both of its partial derivatives vanish at every
smooth point of that component, while the choice

```
x=-G_u(0,v,w)
```

then makes the whole hypersurface singular.  Equivalently, the already
proved smoothness of the plane curve forces its defining polynomial to be
squarefree.

### 1.3 Euler characteristic over an arbitrary algebraically closed field

No embedding of the whole ground field into `C` is required.  Fix a prime
`ell`.  Compactly supported `ell`-adic Euler characteristic over an
algebraically closed characteristic-zero field is additive for a closed-open
decomposition and multiplicative by Kunneth.  It satisfies

```
chi_c(A^n)=1,  chi_c(G_m)=0.
```

Therefore `B x A1 = A4` gives `chi_c(B)=1`, and the strata `u!=0` and `u=0`
give `chi_c(C)=1`.  For a smooth connected affine curve, with smooth
completion of genus `g` and `r>=1` points at infinity,

```
chi_c(C)=2-2g-r.
```

Thus `g=0,r=1`, so `C` is `A1`.  This part of the proof is valid over the
stated ground field.

### 1.4 Abhyankar--Moh--Suzuki

The form needed is the epimorphism/coordinate-line theorem over an
algebraically closed field of characteristic zero:

```
k[v,w]/(g_0) = k^[1], with g_0 irreducible  =>  g_0 is a coordinate.
```

It applies directly, since the preceding steps prove that the *closed plane
embedding* defined by `g_0` is an embedded affine line.  After a plane
automorphism one has `g_0=c v`; rescaling removes `c`.  Hence
`G=v+uK`, and for exponent one the triangular change `x'=x+K` gives
`f=v+ux'`.  No descent of an automorphism chosen over `C` is being silently
used; AMS itself is invoked over `k`.

## 2. Audit of the elliptic-cone cubic corollary

This argument is also correct.

If the cubic part of `f` is independent of `x`, degree considerations give
exactly

```
f=A x^2+xL(y)+G(y),  A in k, deg L<=1, deg G<=3.
```

There are no omitted `x^3`, `x^2 y`, or `x y^2` terms: each would occur in
the cubic homogeneous part.  When `A!=0`, completing the square preserves
the cubic part `H` of `G`.  Each homogenized partial derivative of the new
`G` is a quadric in four variables.  Three homogeneous positive-degree
forms cannot have empty projective zero locus: otherwise their radical
would contain the irrelevant ideal of height four, contradicting the height
bound `<=3` for an ideal generated by three elements.

At infinity their common-zero equations are the three partial derivatives
of `H`.  Euler's identity in characteristic zero shows that such a point
would lie on `H=0`, hence would be a singular point of the plane cubic.
Smoothness excludes it, so the common zero is affine.  Together with
`x'=0` this is a critical point of `f`.  It contradicts the existence of the
transported derivation `D=partial/partial f`, because `D(f)=1` while every
derivation evaluates to zero on `f` at a common zero of its four partials.
The same argument when `L=0`, and the remaining constant/nonconstant cases
of `L`, exhaust all possibilities.

## 3. What remains true for `f=u^m x+G`, `m>=1`

There is a useful exponent-independent consequence which is stronger than
what the two notes currently state.

**Lemma (the special plane divisor is integral and reduced).**  Let

```
f=u^m x+G(u,v,w),  m>=1,
k[x,u,v,w]=B[f].
```

Then `g_0=G(0,v,w)` is a nonconstant irreducible squarefree polynomial.

**Proof.**  The constant cases are excluded exactly as before.  If `g_0=0`,
then `f` is divisible by `u`, contradicting primality of `(f)`.  If
`g_0 in k^*`, then `V_B(u)` is empty, so `u` is a nonconstant unit of `B`.

Write `g_0=c product_i p_i^{e_i}`.  The primes above `(u)` are
`P_i=(u,p_i)`.  Since `B` is regular, `B_(P_i)` is a DVR.  The coefficient
of `P_i` in `div_B(u)` is

```
length_(B_(P_i))(B_(P_i)/u)=e_i;
```

the last equality follows after passing to the generic point of the plane
component, where `B/(u)=k[x,v,w]/(g_0)`.  Thus

```
div_B(u)=sum_i e_i P_i.
```

Factoriality gives `P_i=(h_i)`, and localization at `u` again gives
`h_i=c_i u^{n_i}` in the fraction field.  Valuations at `P_j` yield

```
delta_ij=n_i e_j  for every j.
```

This forces a single factor and then `1=n_1e_1`; hence `e_1=1`.  QED.

The Euler-characteristic calculation also survives unchanged and gives
`chi_c(V(g_0))=1`.  In fact this determines the normalization and the number
of branches.  If `nu:C_tilde -> C` is the normalization, `g` is the genus of
its projective completion, `r_infty` is the number of points at infinity,
and `r_s` is the number of branches over a singular point `s`, additivity for
the finite normalization gives

```
chi_c(C)=2-2g-r_infty-sum_s(r_s-1).
```

All terms subtracted after the initial `2` are nonnegative and
`r_infty>=1`.  Equality `chi_c(C)=1` therefore forces

```
g=0,  r_infty=1,  r_s=1 for every singular s.
```

Thus for every exponent the normalization of the special curve is `A1` and
all its singularities are unibranch.  What fails for `m>=2` is smoothness:
at `u=0` one has

```
partial_u f=G_u(0,v,w),
```

independently of `x`, so a singular point of `g_0=0` need not make the total
hypersurface singular.  An irreducible singular unibranch curve (a cusp is
the basic example) can have compactly supported Euler characteristic one.
This pinpoints the exact opening used by Koras--Russell-type equations.

## 4. Verified strict higher-exponent extensions

Smoothness of the special center is sufficient for **every** exponent.  This
uses the characteristic-zero triviality theorem for affine-plane fibrations
over a smooth curve.

**Theorem (smooth center, every exponent).**  Suppose

```
f=u^m x+G(u,v,w),  m>=1,  k[x,u,v,w]=B[f],
```

and the plane curve `G(0,v,w)=0` is smooth.  Then `f` is a coordinate.

**Proof.**  The divisor lemma, Euler characteristic, and AMS rectify the
special polynomial, so after a plane automorphism and scaling we have

```
G(0,v,w)=v.
```

View

```
B=k[u,x,v,w]/(u^m x+G)
```

as a finitely presented `k[u]`-algebra.  The map `k[u] -> B` is injective,
and `B` is a domain, hence it is torsion-free and therefore flat over the PID
`k[u]`.  Every scheme-theoretic fiber is an affine plane.  Over `u=c!=0`
the equation eliminates `x`, while over `u=0` it becomes `v=0` and
eliminates `v`:

```
B/(u-c)=k[v,w]       (c!=0),
B/(u)=k[x,w].
```

The generic fiber is likewise `k(u)[v,w]`, so this verifies the definition
at all (not only closed) points and after residue-field extensions.

For completeness, the standard triviality input can be decomposed into
precise algebraic steps.  Sathaye's characteristic-zero theorem trivializes
an `A2`-fibration over every rank-one DVR containing `Q`.  Applied at the
local rings of the smooth curve `Spec(k[u])`, and using finite presentation
to spread each local isomorphism, it makes `B` a Zariski locally polynomial
`k[u]`-algebra of relative dimension two.  The Bass--Connell--Wright theorem
identifies such an algebra with the symmetric algebra of a rank-two
projective `k[u]`-module.  That module is free (already because `k[u]` is a
PID), so

```
B=k[u]^[2]=k^[3].
```

Finally the given internal presentation `R=B[f]` makes the two polynomial
generators of `B`, together with `u` and `f`, an ambient coordinate system.
Thus `f` is a coordinate.  QED.

This argument uses exactly flatness, finite presentation, all affine-plane
fibers, a smooth one-dimensional base, and characteristic zero.  It does
not assume in advance that the total space `Spec(B)` is affine space.  The
local fibration theorem is the result of Sathaye/Kambayashi--Wright (also
formulated as Theorem 5(b) of Kraft--Russell, arXiv:1204.3196); the global
linearization step is Bass--Connell--Wright.

For exponent two there is additionally a completely explicit proof.  The
replacement for the exponent-one shear is an exponential automorphism.

**Proposition (constructive smooth-center proof for exponent two).**  Suppose

```
f=u^2 x+G(u,v,w),  k[x,u,v,w]=B[f],
```

and the plane curve `G(0,v,w)=0` is smooth.  Then `f` is a coordinate.

**Proof.**  By the exponent-independent divisor lemma and the
Euler-characteristic calculation, this center is a smooth irreducible
affine curve of Euler characteristic one.  It is therefore `A1`, and AMS
rectifies its plane embedding.  After the resulting plane automorphism and
a scalar normalization,

```
G=v+uK(u,v,w).
```

Write `K=P(v,w)+uH(u,v,w)` and make the shear `x'=x+H`.  It remains to treat

```
f=u^2 x'+v+uP(v,w).
```

On `k[u,v,w,x']` consider the triangular locally nilpotent derivation

```
delta(u)=delta(w)=0,  delta(v)=u,  delta(x')=-P_v.
```

The polynomial `z=u x'+P(v,w)` is in `ker(delta)`.  Hence `z delta` is again
locally nilpotent, and its exponential is a polynomial automorphism.  Since
`delta^2(v)=0`,

```
exp(z delta)(v)=v+z u=v+uP+u^2x'=f.
```

Thus `f` is the image of the coordinate `v`.  QED.

The all-exponent theorem does not include the Koras--Russell cusp mechanism:
its special
plane curve is singular, exactly where the proof stops.

There is also an exponent-independent nonlinear terminal-jet version.  For
`m>=2`, every polynomial

```
f=u^m x+v+u^(m-1)P(v,w)+u^mH(u,v,w)
```

is a coordinate.  After the shear `x'=x+H`, take

```
delta(v)=u^(m-1),
delta(x')=-u^(m-2)P_v,
delta(u)=delta(w)=0,
z=u x'+P.
```

Again `delta(z)=0`, and

```
exp(z delta)(v)=v+u^(m-1)P+u^m x'=f.
```

Finally, the following complementary class controls arbitrary lower
`u`-jets when the dependence on the rectified normal coordinate is linear.

**Proposition.**  Let `m>=1`, and suppose

```
f=u^m x+A(u,w)v+C(u,w)
```

with `A(0,w) in k^*`.  Then `f` is a coordinate of `k[x,u,v,w]`.  In
particular, for arbitrary `m` the cancellation conclusion holds whenever a
smooth center has been rectified to `v=0` and `G` is affine-linear in its
normal coordinate `v`.

**Proof.**  Put `S=k[u,w]`.  After scaling, `A=1+uH`.  The finite geometric
series modulo `u^m` gives `Q,R in S` with

```
A Q+u^m R=1.
```

Hence the row `(u^m,A)` is unimodular over `S`.  Choose `p=R,q=Q` and first
make the translation

```
x -> x+pC,  v -> v+qC.
```

It sends `u^m x+A v` to `u^m x+A v+C`.  The unimodular row is completed by

```
M=[[u^m,A],[-q,p]],  det(M)=u^m p+Aq=1,
```

an element of `SL_2(S)`.  Thus an `S`-linear automorphism of `S[x,v]`
has first coordinate `u^m x+A v`, and composing with the translation has
first coordinate `f`.  Therefore `f` is an ambient coordinate.  QED.

The condition `A(0,w) in k^*` is automatic after AMS if the rectified
`G` is affine-linear in `v`.  This proposition genuinely includes every
exponent `m>=2`.  For `m>=3`, a smooth center can leave nonlinear lower jets
such as `uP_1(v,w)` which the two *explicit automorphism* constructions do
not absorb.  They are nevertheless covered by the affine-plane-fibration
theorem.  Treating them directly as an exponent-one triangular shear would
still be an invalid constructive proof.

## 5. Required edits to the source notes

There is no false implication in either source theorem.  Two presentation
changes are advisable:

1. In the UFD sentence, explicitly insert normality before using
   `Cl(B)=Cl(B[T])`.
2. Replace the broad sentence that higher exponents destroy the
   multiplicity argument.  The divisor argument still proves that `g_0` is
   irreducible and reduced for all `m`; automatic smoothness fails.  The
   final shear fails in general.  For exponent two the LND construction
   above replaces it explicitly, and for every exponent a smooth center is
   handled by the `A2`-fibration argument.
