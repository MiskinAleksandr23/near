# Slice LNDs on `k^[4]`: the full-rank core

Let `R=k[x_1,x_2,x_3,x_4]`, let `D` be a locally nilpotent
`k`-derivation, and suppose that `D(s)=1`.  Put `B=ker(D)`.  Then

```
R=B[s],                 D=partial/partial s,
pi(g)=sum_{j>=0}(-s)^j D^j(g)/j! in B.
```

The second formula is the retraction `R -> B` obtained by setting `s=0`
in the slice coordinates.

## 1. Rank dichotomy

There are no slice LNDs of rank 2 or 3 on `k^[4]`: the rank is either 1
or 4.

Indeed, if the rank is at most 3, then `D(u)=0` for an ambient coordinate
`u` of `R`.  Since `u in B`, for every `c in k`

```
(B/(u-c)B)[s] = R/(u-c)R = k^[3].
```

Cancellation in dimension two gives `B/(u-c)B=k^[2]`.  Thus `B` is an
`A^2`-fibration over `k[u]`; the characteristic-zero triviality theorem
for such fibrations over the affine line gives `B=k[u]^[2]`.  Hence
`R=B[s]=k^[4]` in a coordinate system in which `D=partial_s`, so its rank
is 1.  This is also a direct instance of the result that a regular kernel
of a `k[X_1]`-linear LND on `k[X_1,...,X_4]` is polynomial (Bhatwadekar--
Gupta--Lokhande, arXiv:1501.01438).

Consequently, a counterexample is exactly a rank-4 slice LND.

## 2. Properness is automatic and supplies no extra condition

The action attached to a slice is

```
mu_t(b)=b (b in B),     mu_t(s)=s+t.
```

Its action graph

```
Phi: Ga x Spec(R) -> Spec(R) x Spec(R)
```

is a closed immersion.  On rings, the map

```
R tensor_k R -> R[t]
```

is surjective because `t` is the image of `s_2-s_1`; its image is the
fiber product `Spec(R) x_Spec(B) Spec(R)`, which is closed because
`Spec(B)` is affine and separated.  Hence the action is proper.

Thus the valuative criterion for properness only recovers separatedness
of the already affine quotient `Spec(B)`.  It cannot by itself produce an
invariant ambient coordinate.

The available translation theorems have an additional hypothesis:

* Dubouloz--Finston--Jaradat (arXiv:1303.1032) prove the result for proper
  triangular actions on `A^4` (more generally locally triangulable actions
  on `A^3`-bundles).
* Kaliman (arXiv:1506.06082) proves it for proper actions on `C^4` that
  preserve an ambient coordinate.

The second hypothesis is precisely excluded by rank 4.  A theorem saying
that every proper action on `A^4` with affine quotient is a translation
would already settle the present cancellation problem, since every slice
example has both properties.

## 3. Centralizer calculation

Every derivation commuting with `D=partial_s` has a unique form

```
E = delta + a D,        delta in Der_k(B), a in B,
```

where `delta(s)=0`.  Moreover, `E` is locally nilpotent iff `delta` is
locally nilpotent: on `B`, `E^n=delta^n`, while
`E^n(s)=delta^(n-1)(a)`.  For two such derivations,

```
[delta+aD, epsilon+cD]
  = [delta,epsilon] + (delta(c)-epsilon(a))D.
```

So commuting replicas do not create new transverse directions: the
problem becomes the existence of enough LNDs on the unknown threefold
`Spec(B)`.

There is also a canonical fix-pointed `G_m`-action, generated by the
semisimple derivation `sD`, with grading

```
R = direct_sum_{d>=0} B s^d.
```

Its categorical quotient and fixed locus are both `Spec(B)`.  The
Bass--Haboush structure theorem for smooth fix-pointed reductive actions
says that the quotient is a `G_m`-vector bundle over the fixed locus.  In
this case this only recovers the already explicit trivial line bundle
`Spec(R)=Spec(B) x A^1`.  Linearizing this `G_m`-action would force weights
`(0,0,0,1)` and solve the problem, but that linearization assertion is
again equivalent to proving `B=k^[3]` here.

## 4. A concrete suspension factory for rank-4 candidates

Let `A=k[x,y,z]`, let `delta` be an LND, put `C=ker(delta)=k^[2]`, and
choose a local slice `r` with

```
delta(r)=q in C,        delta(q)=0.
```

For `a in C`, extend to `A[w]` by

```
Delta = delta + a partial_w.
```

The summands commute, so `Delta` is locally nilpotent.  If `(q,a)=C`,
choose `alpha,beta in C` with `alpha q+beta a=1`; then

```
S=alpha r+beta w,       Delta(S)=1,
P=q w-a r,              Delta(P)=0.
```

This gives completely explicit slice LNDs on `k^[4]`.  By the dichotomy,
each is either a translation (rank 1) or a genuine full-rank candidate.
The kernel localizes as

```
(ker Delta)_q = C_q[P].
```

If every unit of `C/(q)` is constant, then every `a` comaximal with `q`
is congruent to a scalar modulo `q`.  In the especially direct case
`a=lambda+hq`,

```
lambda^(-1)(w-h r)
```

is a coordinate slice, so the construction is a translation.  Thus a
nontrivial candidate from this factory must exploit nonconstant units on
the plinth divisor (or a subtler gluing across it).

## 5. An explicit tempting candidate that actually collapses

The Freudenburg rank-3 example is

```
f=xz-y^2,
r=x^3+yf,
g=(f^3+r^2)/x = zf^2+2x^2yf+x^5,
delta=Jac(f,g,-).
```

It satisfies `ker(delta)=k[f,g]`, `delta(r)=-fg`, and has rank 3.  Put
`t=fg` and on `k[x,y,z,w]` define

```
Delta=(t-1)delta+(2t-1)partial_w,
S=4r+(2t-1)w.
```

Because `t` is invariant, `Delta` is LND, and

```
Delta(S)=4[-t(t-1)]+(2t-1)^2=1.
```

This looks like a full-rank slice built from the disconnected divisor
`t(t-1)=0`, but `S` is explicitly a coordinate.  Extend `delta` by
`delta(w)=0`.  The two maps

```
L: w |-> -w+4r                 (fixing A),
E=exp(-(w/2)delta)             (fixing w)
```

are polynomial automorphisms, and

```
(E o L)(w)
 = E(-w+4r)
 = -w+4(r+(t/2)w)
 = 4r+(2t-1)w=S.
```

So this candidate has rank 1.  This calculation is a useful audit test:
Bezout identities on a disconnected plinth divisor can merely encode a
composition of a shear with an exponential automorphism.

## Exact remaining gap in this route

To advance the proper-action route one needs a new argument that forces a
proper slice action on `A^4` to preserve an ambient coordinate.  Properness,
affineness and triviality of the torsor do not do this: they are automatic
from the slice.  To advance the suspension route one needs a pair
`(delta,a)` for which the resulting slice cannot be eliminated by shears
and exponentials, together with a rigorous invariant proving that its
kernel is not `k^[3]`; no such invariant is supplied here.

## 6. The hyperbola plinth target

There is a particularly clean conditional candidate.  Suppose an
irreducible rank-three LND `delta` of `A=k[x,y,z]` has

```
C=ker(delta)=k[u,v],    delta(r)=q=uv-1
```

for a minimal local slice `r`.  On `A[w]` put

```
Delta=delta+u partial_w,
S=-r+v w,
P=(uv-1)w-u r.
```

Then

```
Delta(S)=-(uv-1)+vu=1,     Delta(P)=0,
(ker Delta)_q=C_q[P].
```

Thus `Delta` is a slice LND on `k^[4]`, and the rank dichotomy says that
it is either a translation or has full rank four.  The coefficient `u`
is a genuinely nonconstant unit modulo `q`, with inverse `v`; therefore
the elementary scalar-modulo-`q` elimination does not apply.

Moreover, realization of this plinth ideal would automatically give a
rank-three, rather than rank-two, LND.  The rank criterion for an
irreducible LND on `k^[3]` says that rank `<3` is equivalent to the plinth
ideal containing a nonzero univariate polynomial `F(c)`, where `c` is a
coordinate of `C=k^[2]`.  If `(uv-1)` contained such an `F(c)`, unique
factorization over an algebraically closed field would imply that the
irreducible element `uv-1` is associated to a factor `c-lambda`.  But

```
C/(uv-1) = k[u,u^(-1)]
```

is not `k^[1]`, whereas `C/(c-lambda)=k^[1]`.  This is impossible.

So the exact three-dimensional realization problem is:

> Does there exist an irreducible LND on `k^[3]` whose principal plinth
> ideal, in its polynomial invariant ring, is `(uv-1)`?

The standard rank-three Jacobian constructions do not realize this.
For example, the Freudenburg--Daigle family has local-slice derivative of
the shape `h f^n`; its reduced plinth divisor is a union of intersecting
affine lines, not a copy of `G_m`.  General facts that the invariant ring
is `k^[2]`, the plinth ideal is principal, and `A` is faithfully flat over
the invariant ring do not by themselves forbid a hyperbola divisor.  No
known plinth-divisor theorem used here forces all irreducible components
to be affine lines.  Hence this route currently stops at construction,
not at a structural contradiction.

One useful elimination test for future examples is the following.  If
`delta(r)=q` and a proposed slice has the form

```
S=c r+(lambda+c h q)w,       c,h in C, lambda in k^*,
```

then it is a coordinate: first apply the triangular automorphism
`L(w)=lambda w+c r`, and then `exp(h w delta)`.  Their composition sends
`w` to `S`.  Thus merely being nonconstant modulo a nonreduced plinth
ideal is not enough; higher-order exponential eliminations must also be
excluded.
